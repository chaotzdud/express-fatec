# **API REST Simples com Express e Node.js**

## **1. Para rodar o projeto**

```bash
git clone github.com/chaotzdud/express-api
npm install
npm run dev
```

## **2. Endpoints**

- **POST /users**
- **GET /users**
- **GET /users/:id**
- **DELETE /users/:id**

### Corpo JSON

```json
{
  "id": 1,
  "name": "Eduarda",
  "age": 20   
}
```
