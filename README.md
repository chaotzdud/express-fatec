# **API REST Simples com Express e Node.js**

## **1. Para rodar o projeto**

```bash
git clone github.com/chaotzdud/express-fatec
npm install
node run ./src/server.js
```

## **2. Endpoints**

- **POST /users**
- **GET /users**
- **GET /users/:id**
- **DELETE /users/:id**

### **2.1. Corpo JSON**

```json
{
  "id": 1,
  "name": "Eduarda",
  "age": 20   
}
```
